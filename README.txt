RALFIJS WEB — PROJEKTA PAMĀCĪBA

Faili:
- index.html        — lapas saturs un struktūra
- style.css         — viss dizains, krāsas, izmēri un mobilā versija
- script.js         — mobilā izvēlne, animācijas, gads un kontaktformas darbība
- assets/logo.svg   — caurspīdīgs logo bez fona
- assets/favicon.svg — pārlūka cilnes ikona

KĀ PALAIST
1. Atver mapi ar Visual Studio Code.
2. Instalē Live Server paplašinājumu.
3. Uz index.html spied ar labo peles pogu.
4. Izvēlies “Open with Live Server”.
